Getting Started

Welcome to the Secure Healthcare Crypto System! Our platform provides comprehensive security for healthcare data using industry-standard encryption technologies including AES-256-GCM and RSA-2048. This system is designed for hospitals, clinics, and healthcare providers to securely manage patient data.

Encryption

How to Encrypt Data

  1. Login with your doctor or nurse credentials
  2. Navigate to Services → Encrypt Data from the menu
  3. Click the "Encrypt Data" button
  4. The system will encrypt your data using AES-256-GCM with RSA key wrapping
  5. You'll see a success message with the encrypted payload
Security: All encryption uses hybrid cryptography - AES-256-GCM for data encryption and RSA-2048 for key wrapping, ensuring maximum security for your healthcare data.

Decryption

How to Decrypt Data

  1. Go to Services → Decrypt Data
  2. Select your role (Doctor, Nurse, or Admin)
  3. Click "Decrypt Data"
  4. View the decrypted information based on your access level

Role-Based Access Control (RBAC) ensures that different user roles have appropriate access to sensitive healthcare data.

Secure Email

Sending Secure Emails

  1. Navigate to Services → Send Email
  2. Enter the recipient's email address, or select "Send to All"
  3. Click Send - the email will be encrypted with the recipient's public key
  4. Recipients can decrypt using their private keys

Receiving & Verifying Emails

  1. Go to Services → Receive Email
  2. Enter the sender's email address
  3. Select your role for decryption
  4. The system verifies the digital signature and decrypts the content
